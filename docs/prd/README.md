# EchoIM 产品需求文档

## 1. 产品概述

EchoIM 是一个网页版即时通讯系统，面向中小团队和学习场景，提供完整的 IM 核心能力。

### 1.1 目标用户

- 中小团队内部沟通
- 学习 WebRTC/IM 技术的开发者
- 需要私有化部署 IM 的组织

### 1.2 核心价值

- 开箱即用的 IM 系统，支持单聊/群聊/频道
- 完善的消息能力（撤回/编辑/转发/回复/定时/草稿）
- 语音/视频通话
- 管理后台运营工具

## 2. 功能清单

### 2.1 用户系统

| 功能 | 说明 |
|------|------|
| 注册 | 用户名 + 邮箱 + 验证码 |
| 登录 | 用户名 + 密码，支持邮箱验证码二次验证 |
| 两步验证 | TOTP（Google Authenticator） |
| 可信设备 | 设备指纹识别，免重复验证 |
| 密码找回 | 邮箱验证码重置 |
| 个人资料 | 昵称/头像/性别/签名编辑 |
| 公开主页 | 通过用户名访问 |
| 多账号管理 | 前端支持多账号切换 |

### 2.2 社交关系

| 功能 | 说明 |
|------|------|
| 好友申请 | 发送/接受/拒绝 |
| 好友列表 | 查看/搜索/修改备注 |
| 用户搜索 | 按用户名/昵称搜索 |
| 用户屏蔽 | 屏蔽后双向消息隔离 |

### 2.3 即时通讯

| 功能 | 说明 |
|------|------|
| 单聊 | 1v1 私信 |
| 群聊 | 多人会话 |
| 频道 | 广播式推送（仅群主可发） |
| 文本消息 | 支持 @提及高亮 |
| 图片消息 | 缩略图预览 + 全屏查看器 |
| 文件消息 | 上传/下载 |
| GIF 消息 | 动图发送 |
| 贴纸消息 | 内置贴纸库 |
| 语音消息 | 录制（波形可视化）+ 播放 |
| 消息撤回 | 发送后 2 分钟内可撤回 |
| 消息编辑 | 文本消息可编辑 |
| 消息回复 | 引用原消息回复 |
| 消息转发 | 单条/多条转发到其他会话 |
| 表情回应 | 5 种快捷表情 |
| 消息置顶 | 群聊/频道消息置顶 |
| 消息删除 | 双向删除（发送者可删除双方消息） |
| 定时发送 | 设置未来时间自动发送 |
| 消息草稿 | 自动保存（本地 + 服务端同步） |
| 阅后即焚 | 设置自毁时间（5s/30s/1m/5m/1h） |

### 2.4 消息状态

| 功能 | 说明 |
|------|------|
| 发送状态 | 发送中/发送成功/发送失败 |
| 送达回执 | 单聊消息送达确认 |

### 2.5 会话管理

| 功能 | 说明 |
|------|------|
| 会话列表 | 按最后消息时间排序 |

| 未读计数 | 实时未读消息数 |
| 手动标记未读 | 标记会话为未读 |

### 2.6 群组治理

| 功能 | 说明 |
|------|------|
| 创建群组 | 选择成员创建群聊/频道 |
| 角色体系 | 群主/管理员/普通成员 |
| 群公告 | 编辑公告，变更自动通知成员 |

### 2.7 语音/视频通话

| 功能 | 说明 |
|------|------|

| 视频通话 | 1v1 WebRTC 视频 |

### 2.8 全局搜索

| 功能 | 说明 |
|------|------|
| 统一搜索 | 会话 + 用户 + 消息内容 |
| 关键词高亮 | 搜索结果中关键词高亮 |
| 消息类型筛选 | 按图片/文件/语音/GIF 筛选 |
| 日期范围筛选 | 按日期缩小结果 |
| 跳转到消息 | 点击结果跳转到消息位置 |

### 2.9 安全机制

| 功能 | 说明 |
|------|------|
| JWT 认证 | accessToken + refreshToken |
| 两步验证 | TOTP |
| 可信设备 | 设备指纹 + 授权令牌 |
| 安全事件日志 | 登录异常自动记录 |
| 敏感词过滤 | 拦截/标记两级策略 |
| 文件安全 | Tika 内容检测 + 扩展名白名单 |

| 接口限流 | 防止恶意请求 |

| 功能 | 说明 |
|------|------|
| 数据看板 | 核心指标概览/消息趋势/类型分布/在线统计 |
| 用户管理 | 查询/启用禁用/强制下线 |
| 群组管理 | 查询/解散 |
| 举报管理 | 审核处理（忽略/警告/禁言/封号） |

| 系统公告 | 发布/撤回全员或指定用户公告 |
| 用户封禁 | 临时/永久封禁 + 解除 |

| 版本管理 | 版本发布/灰度/强更 |

## 3. 非功能需求

### 3.1 性能

- 消息列表 CSS content-visibility 优化
- 图片懒加载（IntersectionObserver）
- WebSocket 消息批量处理（requestAnimationFrame）

### 3.2 可观测性

- 健康检查端点
- WebSocket 指标监控
- JVM 指标监控
- 链路追踪（traceId）
- 前端错误上报

### 3.3 兼容性

- 现代浏览器（Chrome/Firefox/Safari/Edge 最新两个版本）
- 响应式布局（桌面 + 平板）
