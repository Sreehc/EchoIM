# EchoIM UI/UX 设计文档

## 1. 设计理念

EchoIM 的设计方向是 **"像 Telegram 但更安静、简约、轻奢"**：

- Serif 标题字体 + 低饱和色板 + 毛玻璃浮层
- 与 Telegram 形成差异化的同时保持 IM 产品的易用性
- 设计系统 Token 化，保证全局一致性

## 2. 设计系统

### 2.1 Token 体系

CSS 变量驱动，分为以下层级：

**字号（7 档）：**
- `--text-2xs` / `--text-xs` / `--text-sm` / `--text-base` / `--text-lg` / `--text-xl` / `--text-2xl`

**圆角：**
- `--radius-sm` / `--radius-md` / `--radius-lg` / `--radius-bubble` / `--radius-pill` / `--radius-panel`

**按钮触控区：**
- `--btn-min-size: 40px`（桌面端）
- 移动端自动升级到 44px

**动效：**
- `--motion-fast` / `--motion-normal` + `--ease-out` / `--motion-ease-out`

### 2.2 色板

采用低饱和色系，深色/浅色双主题：

- 表面色阶：`surface-panel` / `surface-card` / `surface-raised` 三级差异
- 交互色：`interactive-primary-bg` / `interactive-selected-bg` / `interactive-focus-ring`
- 文字色：`text-primary` / `text-secondary` / `text-tertiary` / `text-quaternary`
- 状态色：`status-danger` / `status-warning` / `status-success`

### 2.3 头像调色板

5 个不同色相族，通过用户 ID hash 自动分配，保证相邻用户颜色区分明显。

### 3.2 侧边栏

- 顶部：搜索图标 + 新建按钮 + 文件夹筛选
- 主体：会话列表（置顶项 + 普通项）
- 底部：设置入口

### 3.3 消息气泡

- 己方气泡：右对齐，品牌色背景
- 对方气泡：左对齐，中性色背景
- 气泡内包含：发送者名称（群聊）+ 消息内容 + 时间 + 状态标识
- 扩展元素：回复引用、转发来源、表情回应、置顶徽章、自毁倒计时

### 3.4 通话浮层

- **语音通话**：卡片式 UI，居中显示对方头像/名称/通话时长，底部控制栏
- **视频通话**：全屏远程视频背景 + 本地 PiP（140×186px，右下角）+ 半透明渐变控制栏

## 4. 交互设计

### 4.1 消息发送流程

```
输入框输入 → (可选) 附件/贴纸/@提及/定时/自毁
  → Enter 发送 / 点击发送按钮
  → 气泡出现（sending 状态）
  → ACK 返回（sent 状态）
  → 对方已读（read 状态，双勾变色）
```

### 4.2 搜索交互

```
Cmd/Ctrl + K → 搜索弹窗打开
  → 输入关键词 → 实时搜索（防抖）
  → 结果分三栏：会话 / 用户 / 消息
  → 消息结果关键词高亮
  → 可叠加类型筛选 + 日期筛选
  → 点击结果 → 跳转到对应会话/消息
```

### 4.3 右键菜单

消息气泡右键弹出操作菜单：

- 快捷表情行（5 个常用 emoji）
- 操作列表（复制/回复/转发/置顶/编辑/撤回/删除/举报/已读详情）
- 危险操作红色标记

### 4.4 消息上下文

- **回复**：气泡顶部显示原消息引用条，点击可跳转
- **转发**：气泡顶部显示"转发自 #会话 · 预览"
- **@提及**：消息中 @ 部分高亮，点击跳转到用户资料
- **搜索高亮**：搜索模式下消息中关键词高亮

## 5. 主题系统

### 5.1 深色主题

- 背景：深蓝灰色渐变
- 气泡：半透明毛玻璃效果
- 文字：白色/灰色色阶

### 5.2 浅色主题

- 背景：浅灰白色
- 气泡：白色/浅蓝色
- 文字：深灰色色阶

### 5.3 背景渐变

简化到 4 层（body::before 2 层 + shell 2 层），noise texture 叠加（0.3% opacity paper grain）。

## 6. 响应式设计

- 侧边栏宽度：`clamp(280px, 25vw, 332px)`
- 消息气泡最大宽度：`min(56ch, 68%)`
- 移动端（<768px）：气泡最大宽度 92%
- 搜索弹窗宽度：960px

## 7. 性能优化

- 消息列表和会话列表使用 CSS `content-visibility: auto` + `contain-intrinsic-size` 实现原生渲染优化
- 图片使用自定义指令 `v-lazy-image`（IntersectionObserver + 200px 预加载）
- WebSocket 消息使用 `requestAnimationFrame` 批量处理
- 构建时手动分包：vue-vendor / element-plus

## 8. 无障碍

- 所有交互元素有 `aria-label`
- 右键菜单使用 `role="menu"` / `role="menuitem"`
- 键盘导航支持（Escape 关闭、方向键导航）
- 焦点环使用 `--interactive-focus-ring` 变量
