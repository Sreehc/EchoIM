# EchoIM 用户指南

## 1. 快速开始

### 1.1 注册账号

1. 打开 EchoIM 首页，点击"注册"
2. 输入用户名、邮箱、密码
3. 输入邮箱收到的 6 位验证码
4. 注册成功后自动登录

### 1.2 登录

1. 输入用户名和密码
2. 如开启了两步验证，需输入 TOTP 验证码
3. 勾选"记住我"可在受信设备上免重复验证

## 2. 聊天

### 2.1 发送消息

- **文本消息**：在输入框输入内容，按 Enter 或点击发送按钮
- **图片/文件**：点击输入框左侧的附件按钮，选择文件
- **语音消息**：长按麦克风按钮录制（最长 60 秒），松开发送
- **贴纸**：点击输入框左侧的贴纸按钮，选择贴纸发送
- **GIF**：在附件中选择 GIF 文件发送

### 2.2 消息操作

右键点击消息气泡，可执行以下操作：

- **复制**：复制消息文本
- **回复**：引用原消息回复
- **转发**：转发到其他会话（支持多选）
- **编辑**：编辑自己发送的文本消息
- **撤回**：撤回自己发送的消息
- **删除并撤回**：双向删除消息
- **置顶**：置顶重要消息（群聊/频道）
- **举报**：举报违规消息
- **表情回应**：对消息添加表情反应
- **已读详情**：查看群消息的已读/未读成员（仅自己发送的消息）

### 2.3 定时发送

1. 在输入框编写消息内容
2. 点击输入框左侧的时钟图标
3. 选择日期和时间
4. 点击"定时发送"

待发送消息可在时钟图标旁的面板中查看、取消或立即发送。

### 2.4 消息草稿

在输入框编写消息时，草稿会自动保存。切换到其他会话后再切回来，草稿内容会自动恢复。会话列表中会显示"草稿"标记。

### 2.5 阅后即焚

1. 在输入框编写消息内容
2. 点击输入框左侧的自毁图标
3. 选择自毁时间（5 秒/30 秒/1 分钟/5 分钟/1 小时）
4. 发送后，对方阅读后开始倒计时，到期自动销毁

### 2.6 @提及

在群聊/频道中输入 `@` 会弹出成员选择器，选择成员后消息中会高亮显示。被 @ 的用户会在会话列表中看到 @ 标记。

## 3. 会话管理

### 3.1 会话文件夹

左侧边栏顶部可切换文件夹：

- **收件箱**：所有活跃会话
- **归档**：已归档的会话
- **未读**：有未读消息的会话
- **单聊**：仅单聊会话
- **群聊**：仅群聊会话
- **频道**：仅频道会话

### 3.2 会话操作

右键点击会话项：

- **置顶/取消置顶**：固定到列表顶部
- **免打扰**：静音通知
- **归档**：移入归档文件夹
- **标记未读**：手动标记为未读
- **删除会话**：清除会话记录

### 3.3 Saved Messages

与自己的对话，用于收藏重要消息、文件或笔记。

## 4. 社交功能

### 4.1 添加好友

1. 点击侧边栏的搜索图标
2. 输入用户名搜索
3. 点击"添加好友"，填写申请留言
4. 对方接受后成为好友

### 4.2 全局搜索

按 `Cmd/Ctrl + K` 打开全局搜索：

- 输入关键词搜索会话、用户和消息内容
- 使用消息类型下拉框筛选（图片/文件/语音/GIF）
- 使用日期选择器缩小范围
- 点击消息结果跳转到对应位置

### 4.3 屏蔽用户

在用户资料页点击"屏蔽用户"。屏蔽后：

- 不再收到对方消息
- 双方无法互发消息
- 群聊中消息正常接收但不推送通知

可在设置 → 隐私中管理已屏蔽用户。

### 4.4 举报消息

右键消息 → 举报 → 选择原因 → 提交。

## 5. 群组功能

### 5.1 创建群组

1. 点击侧边栏的"+"按钮
2. 选择"创建群组"
3. 选择群成员
4. 输入群名称
5. 选择群聊或频道类型

### 5.2 群组管理

点击群聊顶部栏的群组名称，打开群组资料面板：

- **编辑群名称**：群主/管理员可修改
- **编辑群公告**：群主/管理员可修改，变更自动通知所有成员
- **成员管理**：查看成员列表、变更角色、禁言、移除
- **邀请链接**：生成邀请链接分享给新成员
- **入群审批**：查看和处理入群申请

### 5.3 成员角色

| 角色 | 权限 |
|------|------|
| 群主 | 全部权限，可解散群组、转让群主 |
| 管理员 | 编辑群信息、管理成员、禁言 |
| 普通成员 | 发消息、查看成员 |

### 5.4 邀请链接

1. 在群组资料面板点击"邀请链接"
2. 设置最大使用次数和过期时间
3. 复制链接分享
4. 新用户打开链接预览群信息后确认加入

## 6. 语音/视频通话

### 6.1 发起通话

在单聊会话中，点击顶部栏的电话图标（语音）或摄像头图标（视频）。

### 6.2 通话中

- **语音通话**：卡片式 UI，显示通话时长
- **视频通话**：全屏远程画面 + 本地画中画（右下角）
- 可切换麦克风/摄像头开关
- 点击挂断按钮结束通话

### 6.3 来电

收到来电时显示来电弹窗，可接听或拒绝。未接来电会在会话中显示系统消息。

## 7. 设置

### 7.1 外观

- 深色/浅色主题切换

### 7.2 聊天偏好

- 回车发送/换行
- 紧凑列表模式
- 紧凑气泡模式

### 7.3 通知

- 浏览器通知权限管理

### 7.4 安全

- 修改密码
- 受信设备管理
- 安全事件日志
- 两步验证（TOTP）管理

### 7.5 隐私

- 已屏蔽用户列表
- 取消屏蔽

## 8. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 打开全局搜索 |
| `Enter` | 发送消息（默认） |
| `Shift + Enter` | 换行 |
| `Escape` | 关闭弹窗/取消编辑 |
| `←` / `→` | 图片查看器切换图片 |
